# SFOC Cases

This folder stores preserved SFOC analysis runs.

Conventions:

- One subfolder per case
- Each case contains:
  - `sfoc_fit_summary.txt`
  - `sfoc_regime_breakpoints.csv`
  - `sfoc_regime_overlay.png`
  - `sfoc_regime_clean_compare.png`
  - `sfoc_oem_diagnostic.csv`
  - `sfoc_oem_diagnostic.txt`
  - `case_manifest.json`
- `case_index.csv` and `case_index.md` summarize all case manifests

The legacy one-off outputs in `analysis/output/` are intentionally left untouched.
